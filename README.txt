Все мапперы, редьюсеры, файлы run, а также датасеты movies.txt, ratings.txt(это изначальные файлы из малого датасета, просто с измененным 
разделителем - запятой) должны быть помещены в локальную директорию, в папку, обозначенную в run как SOURCE_BASE_PATH(изначально - корень).

Далее необходимо выполнить команды:
hadoop dfsadmin -safemode leave 
chmod +x run_all.sh
sh run_all.sh

Также можно отдельно запускать этапы, для этого есть run, run2, run3 соответственно. В случае отдельного запуска также необходимо выполнить 
перед этим chmod +x run3.sh. Запуск 3 части без 2 невозможен, тк там формируется необходимый файл sim.
Итоговые ответы на задачу в файле answers.txt.
